# Code of Conduct

## Unser Standard

Wir wollen ein respektvolles, technisches und konstruktives Umfeld.

Erwartet wird:

- Sachliche Diskussionen.
- Respektvoller Umgangston.
- Kritik an Ideen, nicht an Personen.

Nicht akzeptiert:

- Beleidigungen, Diskriminierung, Drohungen.
- Persoenliche Angriffe oder absichtliches Trollen.
- Veroeffentlichung privater Daten ohne Zustimmung.

## Durchsetzung

Maintainer duerfen Inhalte entfernen, Diskussionen beenden oder Accounts sperren,
wenn Regeln wiederholt verletzt werden.

## Meldung

Bitte melde Vorfaelle vertraulich ueber die in `SECURITY.md` genannte Kontaktadresse.
