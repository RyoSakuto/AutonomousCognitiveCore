# Security Policy

## Supported Scope

Security-relevante Themen in diesem Repo:

- ACC Core (`acc/*`, `main.py`)
- Task-Orchestrierung und Worker-Routing
- Self-Modification Safety-Layer
- Repo-Skripte im Root (`scripts/*`)

## Report a Vulnerability

Bitte keine Security-Luecken als oeffentliche Issue posten.

Sende stattdessen eine vertrauliche Meldung an:

- `security@autonomouscognitivecore.local` (Platzhalter, bei Bedarf anpassen)

Bitte mitliefern:

- betroffene Datei/Komponente
- reproduzierbare Schritte
- erwartetes vs. beobachtetes Verhalten
- Auswirkung (z. B. Datenverlust, Policy-Bypass, RCE)

## Response-Ziele

- Erste Rueckmeldung: innerhalb von 72 Stunden
- Fix-Plan: so schnell wie reproduzierbar machbar
